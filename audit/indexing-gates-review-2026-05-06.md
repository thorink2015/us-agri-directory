## Why this is a one-shot audit, not a drift report

The gate predicates run at build time, so the live noindex meta tag
and the sitemap inclusion are always derived from current data — no
"stale gate" failure mode exists. What this audit produces is a
distribution snapshot you can use to decide whether the **thresholds
themselves** are still appropriate now that operator volume has
risen to 393.

### Threshold tuning question

The threshold of 9 was set when the directory had ~200 operators.
At 393, the median state has ~17 ops. Lowering to 6 would flip
Hawaii and Rhode Island to indexed (small but real states). Lowering
to 5 would only add Rhode Island, since Hawaii is already at 6. No
recommendation — this depends on whether you want every page that
clears 5+ ops to compete for index, or whether 9 reflects a
content-quality floor you want to preserve.
